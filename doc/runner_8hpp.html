Detailed Description

This file provides the tdog::runner class.

Typically, there is no need to use the runner class directly as macros, which access the runner::global() singleton, provide for this. The runner class can be used directly to access lower level functionality, or to have multiple runners. However, there will be a greater degree of support for backward compatibility with macros in the event of future updates.
